# Project Title

This repository is for our final project. It has a couple dev branches so we can upload our progress and eventually combine it all into one QMD file.

## Overview

The purpose of our project was to explore the American National Election Studies (ANES) data. We each picked a subcategory of a topic that we thought would be interesting to see from this data with Daman covering the difference in satisfaction score from how people felt after the 2016 election compared to after the 2024 election, Andrew covering how religious commitment and affiliation shape voting behavior, and with Nick comparing how people said they would vote in the pre-election and post-election surveys to the actual election.
